## 🕹️ About

A dark-themed, **Arcade Noir** personal portfolio inspired by retro gaming aesthetics and modern glassmorphism design. Built as a single-page application with zero frameworks — pure HTML, CSS, and JavaScript — for seamless GitHub Pages deployment.

The design merges **terminal-grade engineering polish** with **8-bit nostalgia** — featuring Pacman-inspired accents, ghost-colored highlights, dot-path section dividers, and neon cyan + purple gradients.

## ✨ Features

| Feature | Description |
|---------|------------|
| 🌌 **Particle Canvas** | Interactive background with mouse-reactive particle network |
| 🕹️ **Pacman Accents** | Chomping loader, dot-path dividers, ghost-colored cards |
| ⌨️ **Typing Effect** | Terminal-style typewriter cycling through roles |
| 🎨 **Glassmorphism** | Frosted glass cards with neon border glow on hover |
| 📊 **Animated Skills** | Progress bars with Pacman indicator arrows |
| 🔢 **Counter Animation** | Stats count up with eased cubic animation |
| 📱 **Fully Responsive** | Mobile-first design with hamburger navigation |
| 🏆 **Project Showcase** | Cards with tech-stack pills and achievement badges |
| ⬆️ **Scroll-to-Top** | Smooth neon button with visibility toggle |
| 🎯 **Active Tracking** | Nav highlights current section on scroll |
